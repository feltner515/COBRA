# PostprocessorSourceScalarKernel

!syntax description /ScalarKernels/PostprocessorSourceScalarKernel

This scalar kernel adds a post-processor value $q(t)$ as a source term for an ODE:
\begin{equation}
  \frac{du}{dt} = q(t) \,.
\end{equation}
