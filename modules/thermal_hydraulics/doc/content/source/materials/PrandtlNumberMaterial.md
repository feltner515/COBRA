# PrandtlNumberMaterial

!syntax description /Materials/PrandtlNumberMaterial

The Prandtl number $Pr$ is computed as:

!equation
Pr = \dfrac{c_p \mu}{k}

with $c_p$ the specific isobaric heat capacity, $\mu$ the dynamic viscosity, and $k$ the thermal
conductivity.
