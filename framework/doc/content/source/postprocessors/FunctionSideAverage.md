# FunctionSideIntegral

This post-processor computes the average $y(t)$ of some function $f(\mathbf{x}, t)$
over a boundary $A$:

!equation
y(t) = \frac{1}{A} \int\limits_A f(\mathbf{x}, t) dA \,.

!syntax inputs /Postprocessors/FunctionSideIntegral
